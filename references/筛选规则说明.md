# 盘前财经资讯筛选规则

> 📌 **核心理念**：每条资讯都先**不做预判**，看它是否含 **A 股影响关键词**。即使标题是"地震""世界杯""俄乌"这种看似无关的内容，只要提到了 A 股相关板块/影响，就保留。

---

## 一、数据源（3 个）

1. **新浪财经** 7×24 快讯（财联社备份源）
2. **东方财富** 快讯
3. **同花顺** 快讯

---

## 二、筛选流程（6 层过滤）

### 第 1 层：始终排除（港股/美股标的）

完全排除，不留余地：

| 类别 | 关键词 |
|------|--------|
| 港股个股 | 港股、恒指、科指、港股通 |
| 美股中概 | 美团、快手、腾讯、拼多多、百度、阿里、京东 |
| 澳门博彩 | 永利澳门、银河娱乐、金沙中国、澳博 |
| 港股金融 | 汇丰、友邦、耐世特 |
| 价格广播 | 长江有色系列、2日X价、X价或 |
| 与投资无关 | 员工薪酬/招聘/校招 |

### 第 2 层：条件性过滤（"看似无关但可能影响 A 股"）⭐ 核心改进

> **新规则**：以前直接排除的类别，现在改成"必须含 A 股影响关键词才放行"。

| 类别 | 关键词 | 放行条件（必须含 A 股影响词） |
|------|--------|-------------------------------|
| **自然灾害/气象** | 地震、台风、洪涝、暴雨、寒潮、高温、干旱 | 保险、灾后重建、种业、农业、煤炭、电力、光伏、化工、建材、应急、运输、港口、航运 |
| **疫情/防控** | 疫情、防控、新冠、核酸、封城、隔离 | A股、股市、消费、旅游、航空、酒店、餐饮、医药、疫苗、外贸、出口、GDP、复苏 |
| **环保/督察/限产** | 环保督察、限产、关停、去产能 | 钢铁、化工、水泥、电解铝、光伏、玻璃、供给侧、涨价、利润 |
| **国际政治/中东/俄乌** | 伊朗、以色列、俄乌、巴以、中东 | 油价、金价、避险、原油、黄金、军工、外贸、出口、航运、天然气 |
| **美联储/欧央行/外盘** | 美联储、鲍威尔、CPI、欧央行、ECB、欧盟 | A股、汇率、美元、人民币、外资、北向资金、中美、出口管制、芯片、开盘 |
| **外国新闻/日韩印** | 日本央行、韩国KOSPI、印度、越南、东盟 | A股、外贸、出口、替代、产业链、汇率、外资、供应链 |
| **活动/会议/峰会** | 发布会、峰会、展览、论坛、开幕 | A股、签约、落地、合作、新政策、订单、采购、项目 |
| **体育赛事** | 世界杯、欧洲杯、奥运会、亚运会、NBA、中超 | 啤酒、体育概念、体育用品、赛事经济、消费、赞助、转播、版权、彩票、李宁、安踏、青岛啤酒 |
| **安全/网安/数据** | 国家安全、网络安全、数据安全、信创 | 启明星辰、深信服、奇安信、安恒信息、天融信、金融科技、数字货币 |
| **反腐/司法** | 反腐、落马、双开、立案审查 | A股、公司、上市、实控人、股权、ST |
| **社会民生/就业** | 就业、失业率、人口、老龄化、出生率 | 消费、养老、医疗、婴童、辅助生殖、房地产、教育 |

**判定示例**：

| 标题 | 结果 | 原因 |
|------|------|------|
| `四川发生6级地震，保险公司已启动应急预案` | ✅ 保留 | 命中"地震"+"保险" |
| `今日晴，最高气温28度` | ❌ 排除 | 命中"天气"但无 A 股影响词 |
| `世界杯开幕，青岛啤酒赞助` | ✅ 保留 | 命中"世界杯"+"青岛啤酒" |
| `世界杯揭幕战，阿根廷 2:0 沙特` | ❌ 排除 | 命中"世界杯"但无 A 股影响词 |
| `美联储宣布加息25个基点，外资或撤离A股` | ✅ 保留 | 命中"美联储"+"A股"+"外资" |
| `美联储会议纪要发布` | ❌ 排除 | 命中"美联储"但无 A 股影响词 |

### 第 3 层：纯行情描述过滤

无机构+无预测词的纯涨跌回顾 → 排除：

- 涨近/涨超/跌近/跌超/涨停/跌停
- 盘中涨/盘中跌/早盘涨/早盘跌/午后涨/午后跌
- 概念回暖/板块反弹/持续走强/异动拉升
- 领涨/走强/活跃

### 第 4 层：机构观点识别

满足 **机构来源 + 预测词 + 投资决策关键词** → 保留为"机构观点"。

**机构来源**：
- 国内券商：中信证券、中信建投、国泰君安、海通证券、华泰证券、招商证券、中金、申万宏源、广发证券、兴业证券
- 国际投行：高盛、摩根士丹利、摩根大通、花旗、瑞银、瑞信
- 行业 KOL：黄仁勋、马斯克、巴菲特、达里奥

**预测词**：预计、有望、目标价、看好、建议、评级、料、认为、称

**投资决策词**：上涨、下跌、买入、增持、减持、目标价、估值、业绩、景气、拐点、赛道、产能、订单、需求

### 第 5 层：自变量模式匹配

满足任一模式 → 保留为"自变量"：

| 类型 | 关键词 |
|------|--------|
| 机构研判 | 后市研判、后市展望、机构看市、券商看市、策略周报、策略月报 |
| 行情预测 | 行情预测、走势预测、有望上涨、有望突破、目标价、看高至、上看、下看、料持续、料上涨、料下跌、走势偏强/偏弱 |
| 板块推演 | 板块轮动、主线、风口、赛道、高景气 |
| 政策前瞻 | 政策预期、政策前瞻、政策红利、政策催化、政策窗口 |
| 行业预判 | 行业拐点、周期见底、景气回升、旺季来临、淡季不淡 |
| 事件催化 | 引爆、驱动、主题投资 |
| 业绩指引 | 业绩预增、业绩超预期、业绩预告 |
| 资金流向 | 资金流入、资金抢筹、北向资金、主力资金、机构调研、机构加仓 |
| 题材推演 | 新基建、国产替代、自主可控 |
| 评级调整 | 买入评级、增持评级、上调评级、首次覆盖、强推 |

### 第 6 层：因变量模式兜底

无机构观点 + 命中回顾性词汇 → 排除：

- 收盘、复盘、今日盘面、今日回顾
- 沪指涨/跌、深成指、创业板指
- 昨日、上周、上月、今年以来
- 累计上涨/下跌、涨幅达、跌幅达
- 数据显示、据统计、历史数据
- 结果出炉、尘埃落定、靴子落地
- 已获批/已通过/已完成/已签署
- 公告、披露、发布、正式、已经

## 三、价值筛选（quality filter）

按来源质量 + 观点强度打分：

- **+3 分**：头部券商 / 国际投行 / 权威媒体 / 监管机构
- **+2 分**：标题含"看好/推荐/买入/增持/目标价"
- **+1 分**：含百分比或"X 亿"等具体数字

按分排序后去除标题相似度 > 90% 的重复内容。

## 五、调整记录

### 2026-06-10 改进

- **核心改进**：将"气象/疫情/国际政治/外国新闻/活动/体育/安全/反腐/民生"等类别从"直接排除"改为"条件性放行"
- 新增 11 个"条件性过滤"类别
- 每个类别配套了 A 股影响关键词字典
- 新增体育赛事的具体 A 股标的（青岛啤酒、李宁、安踏等）
- 新增美联储/欧央行对 A 股开盘价影响的判断

**调整前 vs 调整后**：

| 类型 | 调整前 | 调整后 |
|------|--------|--------|
| `四川地震` | ❌ 直接排除 | ✅ 含"保险"则保留 |
| `世界杯开幕` | ❌ 直接排除 | ✅ 含"青岛啤酒"则保留 |
| `美联储加息` | ❌ 直接排除 | ✅ 含"A股"或"外资"则保留 |
| `俄乌冲突升级` | ❌ 直接排除 | ✅ 含"油价/黄金/军工"则保留 |
